	-> flex-start, flex, end, cnetre, space-evenly, space-around, space-between

25) Flex properties - 
			i)flex-flow - row wrap; (It is used when we want to give both flex-direction and flex-wrap properties together)
			ii)order - The order property specifies the order of the flex items inside the flex container.
			iii)flex-grow - The flex-grow property specifies how much a flex item will grow relative to the rest of the flex items.
					flex-grow:1 //Iska mltb jitna space bcha tha usne usko equally divide krliya 
			iv)flex-shrink - // jiska flex shrink 3 hai wo sabse chota (matlab sabse jyada shrink hua)'
			v) flex-basis- kisi element ko initial size deta h

**Notes - Shorthand for flex properties
